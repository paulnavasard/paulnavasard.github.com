Nesting

Rather than constructing long selector names to specify inheritance, in Less you can simply nest selectors inside other selectors.

  • LESS

              
#menu {
  display: inline-block;

  li {
     a {
      font-weight: bold;
      text-decoration: none;
      color: #F9CB00;

      &:hover {
        color: #FADC11;
        text-decoration: underline;
      }
    }
  }
}

  • Compiled CSS

              
#menu {
  display: inline-block;
}

#menu li a {
  font-weight: bold;
  text-decoration: none;
  color: #F9CB00;
}

#menu li a:hover {
  color: #FADC11;
  text-decoration: underline;
}

Variables

Variables allow you to specify widely used values in a single place, and then re-use them throughout the style sheet, making global changes as easy as changing one line of code.

Comment from a sample css file

      
  /*----------------------------------------------------------
  The base color for this template is #5c87b2. If you'd like
  to use a different color start by replacing all instances of
  #5c87b2 with your new color. EDIT: Actually it's been modified.
  ----------------------------------------------------------*/

  • LESS

              
@heading-color: #FFD100;

h1 {
  color: @heading-color;
  font-size: 2em;
  padding-bottom: 0;
  margin-bottom: 0;
}

h3 {
  color: @heading-color;
  font-size: 1.2em;
}

  • Compiled CSS

              
h1 {
  color: #FFD100;
  font-size: 2em;
  padding-bottom: 0;
  margin-bottom: 0;
}

h3 {
  color: #FFD100;
  font-size: 1.2em;
}

Mixins

Mixins allow you to embed all the properties of a class into another class by simply including the class name as one of its properties.

  • LESS

              
.rounded-corners (@radius: 5px) {
  border-radius: @radius;
  -webkit-border-radius: @radius;
  -moz-border-radius: @radius;
}

#header {
  .rounded-corners;
}

#footer {
  .rounded-corners(10px);
}

  • Compiled CSS

              
#header {
  border-radius: 5px;
  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
}

#footer {
  border-radius: 10px;
  -webkit-border-radius: 10px;
  -moz-border-radius: 10px;
}

Operations

Operations let you add, subtract, divide and multiply property values and colors, giving you the power to create complex relationships between properties.

  • LESS

              
@base-padding: 5px;
@base-color: #333;

header {
  color: @base-color * 2;
  padding-top: @base-padding * 2;
  padding-bottom: @base-padding * 3;
}

footer {
  color: @base-color + #003311;
}

  • Compiled CSS

              
header {
  color: #666;
  padding-top: 10px;
  padding-bottom: 15px;
}

footer {
  color: #336644;
}

You can use brackets in operations

  • LESS

              
@item-height: 50px;
@base-padding: 5px;

.item:first-child {
  height: (@item-height -5) * 2;
  padding: @base-padding (@base-padding * 2);
}

  • Compiled CSS

              
.item:first-child {
  height: 90px;
  padding: 5px 10px;
}

Functions

Functions map one-to-one with JavaScript code, allowing you to manipulate values however you want.

List of Color Functions

  • lighten(@color, 10%);
  • darken(@color, 10%);
  •  
  • saturate(@color, 10%);
  • desaturate(@color, 10%);
  •  
  • fadein(@color, 10%);
  • fadeout(@color, 10%);
  • fade(@color, 50%);

Using the Color Functions

  • LESS

              
@base: #f04615;

.class {
  color: saturate(@base, 5%);
  background-color: lighten(spin(@base, 8), 25%);
}
